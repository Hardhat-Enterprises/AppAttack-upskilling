## What is the Essential 8?

The Essential 8 (E8) is a set of 8 essential cybersecurity controls for organizations to defend against targeted cyberattacks. 

### Controls:

- **Application Whitelisting**
- **Patch Applications**
- **Configure Microsoft Office Macro Settings**
- **User Application Hardening**
- **Restrict Administrative Privileges**
- **Patch Operating Systems**
- **Multi-factor Authentication**
- **Daily Backup of Important Data**

Learn more about how these controls can be implemented to secure your environment.
